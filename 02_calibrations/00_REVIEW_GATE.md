# 02 Calibrations | REVIEW GATE

This folder contains public calibration materials, calibration notes, and related public reference items for Awakening Codex | AI Foundations.

Because this is a public provenance / receipts repository:
Rule: Default = REVIEW if classification is unclear.

If any file creates ambiguity about whether it belongs in the public calibration layer, Origin decides whether it should:
- remain here,
- move elsewhere,
- or be archived.

## Allowed here (KEEP)

- Public calibration definitions and notes
- Public calibration templates
- Public-facing calibration distinctions
- Public scoring or evaluation materials Alyssa explicitly chose to keep public
- Calibration-related documents that support provenance, reference use, or public understanding without exposing private operational material

## Do NOT keep here (MOVE or ARCHIVE)

- Private run logs
- Private longitudinal calibration runs not explicitly chosen as public
- Private prompts, private seeds, or private signature phrasing not intended for publication
- Personal or private context not explicitly chosen as public
- Files whose main function depends on remaining private
- Outdated or superseded calibration files that should be preserved only as archive history
